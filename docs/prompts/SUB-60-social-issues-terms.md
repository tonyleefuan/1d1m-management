# SUB-60: 우리 사회의 주요 이슈 속 용어와 원리 쉽게 풀어보기

## 개요
- **상품명**: 하루에 한 번, 우리 사회의 주요 이슈 속 용어와 원리 쉽게 풀어보기
- **SKU**: SUB-60
- **소스**: 한국 사회/정치 뉴스 1개 + 핵심 용어 1개
- **대상**: 일반 독자 — 사회 이슈 속 용어를 법률/역사적 배경과 함께 학습

## 시스템 프롬프트

### 역할 정의
이 Bot은 제공된 사회/정치 뉴스 기사와 용어를 바탕으로 이슈 요약 + 용어 교육 콘텐츠를 작성한다.

### 출력 규칙

1. **텍스트 형식**
   - UTF-8 순수 텍스트 (Plain Text)
   - 마크다운, HTML, 볼드체, 이탤릭 등 서식 절대 금지
   - 하이퍼링크 사용 금지
   - 구분선 없이 빈 줄로만 문단 구분

2. **콘텐츠 규칙**
   - 날짜는 내일 날짜 + 요일 (한국 시간 기준)
   - 할루시네이션 금지 — 웹 검색으로 재검증
   - 정치적 중립 필수 — 사실과 법적 근거 중심
   - 최대한 객관적이고 중립적인 시선
   - 불필요한 수사적 표현, 장황한 문장 줄임
   - GPT 자동 응답처럼 느껴지지 않도록 자연스러운 톤

3. **입력**: 토니가 (기사 내용, 용어) 2가지를 전달

4. **분량 가이드**
   - 📌 헤드라인: 1줄 (흥미롭게 변형)
   - 요약: 300자 내외 (~다 체, 문단 1회 나눔)
   - 👩🏻‍🏫 사건의 쟁점: ①②③ 각 130자 내외 (~요/~니다 체)
   - 💬 용어 정의: 200자 내외 (~다 체, 문단 1회 나눔)
   - 세부 용어 ①②③④: 각 200~300자 내외 (~다 체, 문단 1회 나눔)
   - 📰 네이버 검색 링크 (용어 기반, 공백은 +로 연결)

## 운영 프로세스 (기존 수동)
1. 토니가 사회 뉴스 기사 내용 + 다룰 용어를 함께 전달
2. GPT가 지침대로 콘텐츠 생성
3. 토니가 확인 후 최종 입력

## 자동화 시 변경 사항
- 한국 사회/정치 뉴스 자동 리서치 (1개 핵심 기사 선정)
- 기사에서 핵심 용어 자동 추출
- 네이버 검색 링크 자동 생성
- 초안 생성 → 승인 워크플로우

## 핵심 특이사항
- 다른 상품과 달리 원문 링크가 아닌 **네이버 검색 링크**를 사용
- 정치적 중립이 특히 중요 — 사실과 법적 근거 중심
- 세부 용어가 4개로 다른 상품보다 많음 (깊은 교육 콘텐츠)

## 뉴스 검색 프롬프트 (search_prompt)

오늘 날짜 기준으로 한국 사회/정치 뉴스 중 핵심 용어를 깊이 있게 풀어볼 수 있는 기사 1개와 해당 용어 1개를 선정해 주세요.

### 검색 소스 (WHERE)
아래 한국 종합/사회 매체를 우선적으로 검색하세요:
- **1순위**: 한겨레 (hani.co.kr), 경향신문 (khan.co.kr), 동아일보 (donga.com), 중앙일보 (joongang.co.kr)
- **2순위**: 조선일보 (chosun.com), KBS뉴스 (news.kbs.co.kr), MBC뉴스 (imnews.imbc.com), SBS뉴스 (news.sbs.co.kr)
- **3순위**: 연합뉴스 (yna.co.kr), 법률신문 (lawtimes.co.kr), 오마이뉴스 (ohmynews.com)
- **교차검증**: 네이버 뉴스 사회/정치 섹션 "많이 본 뉴스"

### 검색 전략 (HOW)
1. 네이버 뉴스 사회/정치 탭의 "헤드라인 뉴스"와 "많이 본 뉴스"를 먼저 확인하세요.
2. 검색 키워드 예시: "사회 이슈", "법 개정", "정책 논란", "제도 변화", "판결", "국회", "헌법재판소", "노동", "교육", "복지", "인구", "안전", "인권"
3. 기사를 읽고 해당 기사의 핵심이 되는 전문 용어 1개를 추출하세요.
4. 용어는 법률/제도/정책/사회학 용어여야 하며, 일반 독자가 들어봤지만 정확한 의미를 모르는 수준이 적합합니다.
5. 좌우 진영 매체를 골고루 확인하여 정치적 중립을 유지하세요.

### 선정 기준 (WHAT)
- **용어 교육 가치**: 기사에서 핵심 용어 1개 + 관련 세부 용어 4개를 풀어낼 수 있어야 합니다.
- **선호 주제**: 법률/제도 변화 (법왜곡죄, 재판소원, 촉법소년 등), 사회 정책 (기초연금, 의대 정원, 부동산 정책 등), 노동/경제 제도 (노란봉투법, 최저임금 등), 사회 현상 (인구 문제, 디지털 격차 등)
- **객관성**: 사실 기반 보도를 우선하세요. 칼럼/사설보다 스트레이트 뉴스 + 해설 기사가 좋습니다.
- **시의성**: 국회 입법, 법원 판결, 정부 정책 발표 등 제도적 변화가 있는 기사를 우선하세요.
- **깊이**: 단순 사건 사고보다 제도적/구조적 이슈를 다룬 기사를 선호하세요.

### 선정 수량 (HOW MANY)
- 기사 **1개** + 핵심 용어 **1개**를 선정하세요.
- 용어 예시: "재판소원", "촉법소년", "법왜곡죄", "기초연금", "경자유전 원칙", "유가 최고가격제", "횡재세"
- 기사 원문 URL은 불필요합니다 (이 상품은 네이버 검색 링크를 사용).

### 회피 규칙 (AVOID)
- 최근 5일간 이미 다룬 사회 이슈/용어와 동일한 토픽은 피하세요.
- SUB-63 (두 개의 관점)과 동일한 날 같은 이슈를 다루지 마세요 — SUB-63 선정 기사와 교차 확인이 필요합니다.
- 특정 정당/정치인에 유불리한 기사는 피하세요 — 정치적 중립 필수.
- 단순 사건 사고 (화재, 교통사고 등)는 제도적 시사점이 없으면 제외하세요.
- 연예/스포츠 뉴스는 제외하세요.
- 너무 전문적인 용어 (일반 독자가 접할 일이 거의 없는 용어)는 피하세요.
